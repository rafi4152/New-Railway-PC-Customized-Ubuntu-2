# 🖥️ Ubuntu 24.04 Desktop on Railway.app (noVNC)

এই রিপোজিটরির মাধ্যমে আপনি খুব সহজেই **Railway.app**-এ একটি সম্পূর্ণ, লাইটওয়েট এবং ফাস্ট **Ubuntu 24.04 (Noble Numbat)** ডেস্কটপ সেটআপ করতে পারবেন। কোন থার্ড-পার্টি সফটওয়্যার (যেমন: AnyDesk বা TeamViewer) ছাড়াই শুধুমাত্র ব্রাউজারের মাধ্যমে **noVNC** দিয়ে এই পিসিটি কন্ট্রোল করা যাবে।

---

## ✨ ফিচারসমূহ

- 🚀 **লাইটওয়েট ডেস্কটপ:** দ্রুত পারফরম্যান্সের জন্য **XFCE4** ডেস্কটপ এনভায়রনমেন্ট ব্যবহার করা হয়েছে।
- 🌐 **ব্রাউজার এক্সেস:** **noVNC** ইন্টিগ্রেশনের কারণে যেকোনো ডিভাইস (পিসি, মোবাইল, ট্যাবলেট) এর ব্রাউজার থেকে সরাসরি এক্সেস করা যায়।
- ⚙️ **সহজ ডিপ্লয়মেন্ট:** জাস্ট গিটহাব রিপোজিটরি কানেক্ট করে Railway-তে ডিপ্লয় করলেই অটোমেটিক সবকিছু রেডি হয়ে যাবে।
- 🐧 **লেটেস্ট ওএস:** Ubuntu 24.04 বেইজড একদম ফ্রেশ একটি সেটআপ।

---

## 🛠️ ডিপ্লয়মেন্ট গাইড (কিভাবে Railway-তে রান করবেন)

**ধাপ ১: রিপোজিটরি ফর্ক বা তৈরি করা**
প্রথমে এই ফাইলগুলো (`Dockerfile` এবং `start.sh`) দিয়ে আপনার নিজের গিটহাব একাউন্টে একটি রিপোজিটরি তৈরি করুন। 

**ধাপ ২: Railway-তে প্রজেক্ট তৈরি**
1. [Railway.app](https://railway.app/)-এ লগইন করুন।
2. ড্যাশবোর্ড থেকে **"New Project"** বাটনে ক্লিক করুন।
3. **"Deploy from GitHub repo"** সিলেক্ট করুন এবং আপনার তৈরি করা রিপোজিটরিটি বেছে নিন।
4. Railway অটোমেটিকভাবে `Dockerfile` ডিটেক্ট করে প্রজেক্টটি বিল্ড করা শুরু করবে।

**ধাপ ৩: ডোমেইন জেনারেট করা**
1. বিল্ড এবং ডিপ্লয় কমপ্লিট হওয়ার পর প্রজেক্টের সার্ভিসটিতে ক্লিক করুন।
2. উপরের মেন্যু থেকে **Settings**-এ যান।
3. একটু নিচে স্ক্রল করে **Networking** সেকশন থেকে **"Generate Domain"**-এ ক্লিক করুন।
4. আপনাকে একটি পাবলিক লিংক বা ডোমেইন দেওয়া হবে।

**ধাপ ৪: পিসিতে কানেক্ট করা**
1. জেনারেট করা ডোমেইন লিংকটিতে ক্লিক করলে ব্রাউজারে noVNC-এর পেজ ওপেন হবে।
2. **Connect** বাটনে ক্লিক করুন।
3. পাসওয়ার্ড চাইলে নিচে দেওয়া ডিফল্ট পাসওয়ার্ডটি দিন।
