# 🖥️ Ubuntu 24.04 Desktop on Railway.app (noVNC)

এই রিপোজিটরির মাধ্যমে আপনি খুব সহজেই **Railway.app**-এ একটি সম্পূর্ণ, লাইটওয়েট এবং ফাস্ট **Ubuntu 24.04 (Noble Numbat)** ডেস্কটপ সেটআপ করতে পারবেন। কোন থার্ড-পার্টি সফটওয়্যার ছাড়াই শুধুমাত্র ব্রাউজারের মাধ্যমে **noVNC** দিয়ে এই পিসিটি কন্ট্রোল করা যাবে।

**আপডেট:** এই ভার্সনে noVNC-এর কানেকশন ইস্যু (`Échec de connexion au serveur`) ফিক্স করা হয়েছে এবং পুরো সিস্টেমের ভাষা ডিফল্টভাবে **English (US)** সেট করা হয়েছে।

---

## ✨ ফিচারসমূহ

- 🚀 **লাইটওয়েট ডেস্কটপ:** দ্রুত পারফরম্যান্সের জন্য **XFCE4** ডেস্কটপ এনভায়রনমেন্ট।
- 🌐 **ব্রাউজার এক্সেস:** **noVNC** এর মাধ্যমে যেকোনো ডিভাইসের ব্রাউজার থেকে সরাসরি এক্সেস।
- 🇺🇸 **English Default:** সার্ভার এবং noVNC উভয়ের জন্যই `en_US.UTF-8` ভাষা ফিক্স করে দেওয়া হয়েছে, যাতে ফ্রেঞ্চ বা অন্য কোনো ভাষা শো না করে।
- 🔌 **কানেকশন ফিক্সড:** Railway পোর্টের সাথে `0.0.0.0` বাইন্ড করা হয়েছে, ফলে আর কোনো কানেকশন এরর আসবে না।
- 🐧 **লেটেস্ট ওএস:** Ubuntu 24.04 বেইজড একদম ফ্রেশ একটি সেটআপ।

## 🛠️ ডিপ্লয়মেন্ট গাইড (কিভাবে Railway-তে রান করবেন)

**ধাপ ১: গিটহাব রিপোজিটরি তৈরি**
প্রথমে এই রিপোজিটরির ফাইলগুলো (`Dockerfile` এবং `start.sh`) আপনার নিজের গিটহাব একাউন্টে আপলোড করুন বা ফর্ক করুন।

**ধাপ ২: Railway-তে প্রজেক্ট তৈরি**
1. [Railway.app](https://railway.app/)-এ লগইন করুন।
2. ড্যাশবোর্ড থেকে **"New Project"** বাটনে ক্লিক করুন।
3. **"Deploy from GitHub repo"** সিলেক্ট করুন এবং আপনার রিপোজিটরিটি বেছে নিন।
4. Railway অটোমেটিকভাবে প্রজেক্টটি বিল্ড করা শুরু করবে।

**ধাপ ৩: ডোমেইন জেনারেট করা**
1. বিল্ড কমপ্লিট হওয়ার পর প্রজেক্টের সার্ভিসটিতে ক্লিক করুন।
2. **Settings**-এ যান।
3. **Networking** সেকশন থেকে **"Generate Domain"**-এ ক্লিক করুন।

**ধাপ ৪: পিসিতে কানেক্ট করা**
1. জেনারেট করা ডোমেইন লিংকটিতে ক্লিক করলে ব্রাউজারে noVNC-এর পেজ ওপেন হবে।
2. **Connect** বাটনে ক্লিক করুন।
3. পাসওয়ার্ড চাইলে নিচের ডিফল্ট পাসওয়ার্ডটি দিন।
